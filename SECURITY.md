# Security Policy

## Supported Versions

Only the latest version of Silicon Epoch is actively supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| v1.0.x  | :white_check_mark: |
| < v1.0  | :x:                |

## Reporting a Vulnerability

If you discover a potential security vulnerability in this project, please report it responsibly by emailing the project maintainer at **iarhaaan@gmail.com**.

Do not open a public GitHub issue for security vulnerabilities. We will investigate and respond to all reports promptly.
